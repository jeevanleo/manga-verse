## Installation and Setup

Follow these steps to clone the project and set it up:

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/jeevanleo/manga
   cd manga
   ```

2. **Install Dependencies:**

   ```bash
   npm install
   ```

3. **Start the Backend:**

   ```bash
   cd Backend
   npm start
   ```

4. **Start the Frontend:**

   ```bash
   cd Frontend
   npm run dev
   ```
